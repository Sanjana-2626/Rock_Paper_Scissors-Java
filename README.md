## Description
This is a simple Java console-based Rock Paper Scissors game.
The user selects rock, paper, or scissor, and the computer randomly selects one option.
The program determines the winner based on standard game rules.

## Technologies Used
- Java
- Scanner class
- Random Class
- Conditional statements

## How to Run
1. Compile the program:
   javac RockPaperScissor.java
2. Run the program:
   java RockPaperScissor

## Features
- User input validation
- Random computer choice
- Displays game result clearly
